# Jazz Scale Generator (Python)

This program generates musical scales based on user input.

## Features
- Enter a musical key to generate its scale
- Supports multiple common keys used in jazz and music
- Displays notes in correct order

## Example
Enter key: Bb  
Output: Bb C D Eb F G A  

## How to Run
1. Install Python
2. Run:
   python jazz_scale_generator.py

## Skills Used
- Python dictionaries
- User input handling
- Conditional logic
- Combining music theory with programming
